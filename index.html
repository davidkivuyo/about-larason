CloudPanel

CloudPanel is a next-generation hosting control panel designed for raw speed, top-tier security, and server simplicity. It enables anyone to configure high-performance servers for PHP, Node.js, Python, and static sites in just a few clicks.

Through host.larason.space, we offer a completely free control panel experience. Create databases, set up secure SSH/SFTP access, add subdomains, schedule cron jobs, and install Let's Encrypt SSL certificates without any expensive panel license fees.

Privacy Search

SearXNG

searxng.larason.space

SearXNG is a privacy-respecting, hackable metasearch engine that aggregates results from over 70 search engines and databases. Crucially, SearXNG does not track search history, log user IP addresses, or build profiling files.

Our public instance filters out tracking cookies and identifiers, returning completely unbiased, clean search results. Query pages across web search, images, videos, news, maps, and programming files safely, free from commercial profiling.

Web Editor

Codeer

codeer.larason.space

Codeer is a sleek, responsive, browser-based editor that turns any device into a lightweight development sandbox. It features rich syntax highlighting, auto-completion, and file management tools designed for developers on the move.

Write, edit, and experiment with Javascript, HTML, CSS, or shell scripts instantly. Because it runs directly in your browser, you bypass complex installations—making it the ideal free tool for classroom learning, rapid prototyping, and quick edits.

About Larason LLC

Larason LLC is an independent, non-commercial pioneer dedicated to deploying state-of-the-art open-source platforms for public benefit.

We believe that modern technology should be democratized. Too often, excellent open-source applications are either locked behind complex command-line setup routines, or centralized by commercial providers who profit off your activity and data.

By hosting pre-configured, production-ready instances of Forgejo, CloudPanel, SearXNG, Codeer, and wikis, we eliminate the deployment friction. We handle server administration, automated backups, SSL maintenance, and scaling, giving you premium-tier software completely free of cost.
